## 🔬 杜邦分析计算逻辑

### 1. 核心公式

根据杜邦分析的经典三层拆解模型：

```
ROE（净资产收益率）= 销售净利率 × 总资产周转率 × 权益乘数
```

### 2. 指标定义与计算

| 指标 | 计算公式 | 业务含义 |
|------|----------|----------|
| **销售净利率** | 净利润 / 营业收入 | 衡量盈利能力，每元收入创造的利润 |
| **总资产周转率** | 营业收入 / 平均总资产 | 衡量运营效率，资产的利用效率 |
| **权益乘数** | 平均总资产 / 平均股东权益 | 衡量财务杠杆，负债经营程度 |

### 3. 三层拆解逻辑

```
第一层：ROE = 总资产净利率 × 权益乘数
    ↓
第二层：总资产净利率 = 销售净利率 × 总资产周转率
    ↓
第三层：销售净利率 = 净利润 / 营业收入
        总资产周转率 = 营业收入 / 平均总资产
        权益乘数 = 1 / (1 - 资产负债率)
```

### 4. 计算流程（代码实现）

```python
# 核心计算逻辑 - 摘自 dupont_analysis.py
def calculate_roe(self, net_profit_ratio, asset_turnover, equity_multiplier):
    """计算ROE = 销售净利率 × 总资产周转率 × 权益乘数"""
    if None in [net_profit_ratio, asset_turnover, equity_multiplier]:
        return None
    return net_profit_ratio * asset_turnover * equity_multiplier
```

```python
def convert_value(self, value):
    """智能单位转换：10亿 → 1000000000"""
    match = self.pattern.match(value)
    if match:
        num = float(match.group(1))
        unit = match.group(2)
        if unit == '亿':
            return num * 100000000
        elif unit == '万':
            return num * 10000
```
